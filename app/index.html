Come usare UniNote
1
Crea il backup su ogni dispositivo
Apri JW Library → icona ☁️ Nuvola → Crea backup → salva il file .jwlibrary. Fallo su tutti i dispositivi che vuoi sincronizzare.
2
Carica i file in UniNote
Torna qui e tocca Seleziona file… Carica almeno 2 file .jwlibrary. Vedrai subito quante note, evidenziazioni e segnalibri ha ciascuno.
3
Confronta e risolvi i conflitti
Se la stessa nota è stata modificata su due dispositivi diversi, UniNote te la mostra. Puoi scegliere: tieni A, tieni B, tieni entrambe o unisci il testo.
4
Unisci e scarica
Tocca Unisci e scarica. UniNote crea un file unico completo, senza duplicati, con tutto il meglio di ogni dispositivo.
5
Ripristina in JW Library
Apri JW Library → icona ☁️ Nuvola → Ripristina backup → seleziona il file scaricato. Fatto!
Domande frequenti
🔒 I miei file vengono caricati su internet?
No. UniNote funziona interamente sul tuo dispositivo. I tuoi dati non escono mai dal browser.
📱 Funziona tra iPhone, Android e Windows?
Sì! Il formato .jwlibrary è identico su tutti i sistemi. Puoi unire backup di dispositivi diversi senza problemi.
⚠️ Posso perdere dati?
UniNote non modifica mai i file originali. Crea sempre un nuovo file separato. I tuoi backup restano intatti.
